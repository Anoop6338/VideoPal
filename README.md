        <h1>VideoPal</h1>
        <p><strong>VideoPal</strong> is an Android application that allows users to engage in random video calling with strangers. It provides a platform for spontaneous and exciting connections, enabling users to have video conversations with people from around the world.</p>
        <p>The app utilizes <strong>WebRTC</strong> (Web Real-Time Communication) technology for seamless peer-to-peer video and audio communication. With its intuitive user interface and robust functionality, VideoPal provides a unique and immersive experience for users to meet and interact with new people through video calls.</p>

        <h2>Features</h2>
        <ul>
            <li><strong>Random Video Calling:</strong> Connect with random strangers through live video calls.</li>
            <li><strong>Real-Time Communication:</strong> Utilize WebRTC for high-quality, low-latency peer-to-peer video and audio communication.</li>
            <li><strong>User Profiles:</strong> Create personalized profiles with profile pictures and basic information.</li>
            <li><strong>Call Accept/Reject:</strong> Accept or reject incoming video call requests from strangers.</li>
        </ul>

        <h2>Technologies Used</h2>
        <ul>
            <li><strong>Android:</strong> Java, XML</li>
            <li><strong>WebRTC:</strong> Web Real-Time Communication for video and audio streaming</li>
            <li><strong>Firebase:</strong> Authentication, Realtime Database, Storage</li>
            <li><strong>Glide:</strong> Image loading and caching library for smooth image rendering</li>
            <li><strong>PermissionsDispatcher:</strong> Simplify runtime permission handling in Android</li>
            <li><strong>Retrofit:</strong> HTTP client for API requests</li>
            <li><strong>GSON:</strong> JSON parsing library for data serialization/deserialization</li>
        </ul>
